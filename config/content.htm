建立個人倉儲cd2021

1.下載kmol2021_spring_v2.7z，完成後開啟。

2.登入個人github帳號，創建新的倉儲cd2021。

3.進入tmp。

4.輸入git clone --recurse-submodules https://github.com/40623121/cd2021.git 取得子模組資料。

5.完成後，進入cmsimde資料夾輸入python wsgi.py開始編輯倉儲。

6.於瀏覽器輸入https://localhost:9443/

7.編輯完成後，輸入git status檢查 -> git add .全部新增 -> git commit -m "項目名稱" -> git push推送。
